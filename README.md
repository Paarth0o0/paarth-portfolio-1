🚀 Paarth Portfolio
A modern, responsive, and interactive personal portfolio website built using HTML, CSS, and JavaScript. Designed with a sleek glassmorphism-inspired interface, smooth animations, responsive layouts, and dark/light theme support, this portfolio serves as a professional platform to showcase my skills, projects, experience, and technical journey.

🌟 Overview
This project was created to establish a professional online presence while demonstrating front-end development skills and modern web design principles.

As a passionate and cross-domain software developer currently pursuing a Bachelor of Computer Applications (BCA), I enjoy building solutions that bridge multiple areas of technology, including Full-Stack Development, Artificial Intelligence, Cybersecurity, Cloud Computing, DevOps, and Automation.

This portfolio serves as a central hub for showcasing my technical skills, projects, academic journey, and continuous learning path. It reflects my commitment to growth, hands-on experimentation, and building real-world solutions through modern technologies.
